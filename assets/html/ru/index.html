Welcome to Madsonic

With Madsonic you can easily stream or download music from your home computer to your Android phone (and do lots of other cool stuff too).

To install the Madsonic server software on your computer, please visit madsonic.org.
Ther server software is free and it's available for Windows, Mac, Linux and Unix.

By default, this program is configured to use the Subsonic demo server

Once you've set up your own server, please go to Settings and change the configuration so that it connects to your own computer.

Madsonic Server Features

  • Artist Hub.
  • Similar Artist Search.
  • Share with your own Domain.
  • Mediafile Access Control.
  • Genre Radio Player.
  • improved Playlist Handling.
  • advanced Media Scanning.
  • improved Media Search.
  • improved Last.FM Integration
  • Left & Playqueue Frames resizable.
  • Players Top/played Songs Statistic.
  • Youtube Songsearch Feature.
  • extended User History Logger.
  • extended DBViewer.
  • extended Mediatypes.
  • extended Albumset visualization.
  • Madsonic featured Themes.
  • Custom Scrollbar.
  • Custom Accordion.
  • No ads in the Madsonic web interface.
  • Unlimited streaming and download to any number of Android phones.
Other features to be released later.
